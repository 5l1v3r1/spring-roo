Roo really represents productivity without compromise. ## Quick Start ### Pre-requisites To start with Spring Roo you need: * [JDK 1.6 or JDK 1.7]( http://www.oracle.com/technetwork/java/javase/downloads/index.html) * [Maven 3.2 or above]( http://maven.apache.org/download.html ) * Internet access so that Maven can download required dependencies ### Working with Spring Tool Suite (STS) Depending on the version of Spring Roo you are using you must install the appropriate STS: * Install STS 3.7.0 to work with Spring Roo 2.x. Go to [Spring Roo 2 Reference Documentation]( http://docs.spring.io/spring-roo/docs/2.0.0.M1/reference/html/#install-roo-support ) and follow the instructions. * Install STS 3.6.2 to work with Spring Roo 1.x. Go to [Spring Roo 1 Reference Documentation]( http://docs.spring.io/spring-roo/docs/1.3.1.RELEASE/reference/html/usage.html#install-roo-support ) and follow the instructions. ### Running from shell {%
